- Keep `index.html` as the GitHub Pages landing page.
- Keep `sitemap.xml` and `robots.txt` in the repo root.
- Share the GitHub Pages URL in SoundCloud descriptions.
- Share the GitHub Pages URL in Facebook posts.
- Link back to this repo from FreeEQ8, Voxel Audio, and the TizWildinEntertainmentHUB.
- Add a GitHub social preview image under repo **Settings → Social preview**.

## Sample pack repo link rule

Every free sample-pack row must include:

- pack name
- lane / genre
- description
- direct dedicated GitHub repo link where verified
- Hub link as the central catalog fallback

Do not invent unverified repo URLs. Use the Hub repo/catalog route until a standalone repo exists or is confirmed.
